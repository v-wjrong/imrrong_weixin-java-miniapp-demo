| Brief Description | This is a Spring Boot-based backend service module for WeChat Mini Programs. It serves as a bridge between business applications and the WeChat ecosystem, providing unified API proxy and key business logic encapsulation. Core functionalities include file upload management, user login authentication, and server message processing. The module adheres to RESTful specifications, dynamically loads Mini Program configurations during requests, and relies on Spring Boot and the WeChat Java SDK for implementation. |
